Este README proporciona una guía paso a paso para instalar y utilizar el proyecto TechStore. Siga estos pasos en orden para evitar errores y garantizar una instalación correcta.

Creación de tablas: Ejecuta el script de creación de tablas para establecer la estructura básica de la base de datos.

Funciones y Stored Procedure: Ejecuta el script de funciones y stored procedures.

Vistas: Ejecute el script de vistas para facilitar la visualización de datos.

Triggers: Ejecuta el script de triggers

Inserción de datos: Ejecuta el script de inserción de datos para proporcionar ejemplos iniciales en la base de datos. Asegúrese de que esta etapa se realice después de crear las tablas para evitar conflictos de claves primarias.
